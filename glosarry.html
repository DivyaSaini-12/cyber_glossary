Cyber Security Glossary

Firewall

Unwanted traffic ko block karta hai.

Malware

Harmful software jaise virus, worm, trojan.

Phishing

Fake email/website se info churana.

Antivirus

Malware detect aur remove karta hai.

Encryption

Data ko unreadable format me convert karna.

Decryption

Encrypted data ko wapas normal banana.

DDoS Attack

Server ko overload karke crash karna.

VPN

Private aur secure internet connection banata hai.

Two-Factor Authentication

Login ke liye 2 step ka protection.

Brute Force Attack

Password guess karne ki technique.

Keylogger

User ke typed keys ko secretly record karta hai.

Ransomware

Files lock karke ransom mangta hai.

Spyware

Bina permission ke user activity track karta hai.

Social Engineering

Logo ko fool karke unka data lena.

Cybersecurity

Online threats se data aur systems ki suraksha.

SQL Injection

Database me malicious query daalna.

Zero-Day

Naya discovered security weakness.

Botnet

Hacked devices ka group jo ek hacker control karta hai.

Penetration Testing

Security test jisme system me ghusne ki koshish hoti hai.

IDS

Suspicious activity detect karta hai.

IPS

Attack hone se pehle rokta hai.

HTTPS

Secure version of HTTP.

Honeypot

Fake system jo attackers ko trap karta hai.

Backdoor

Hidden way to access system.

Digital Signature

Electronic form of signature for authenticity.

Authentication

User identity verify karna.

Authorization

Access rights dena.
